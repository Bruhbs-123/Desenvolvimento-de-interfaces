# 🎨 Desenvolvimento de Interfaces

## 📚 Sobre o projeto

Este repositório reúne as atividades desenvolvidas na disciplina de **Desenvolvimento de Interfaces**, com foco na criação de páginas web modernas, organizadas e interativas.

Durante a matéria, foram aplicados conceitos de HTML, CSS e JavaScript para desenvolver interfaces com foco na experiência do usuário (UX).
